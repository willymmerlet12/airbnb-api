## Install and Usage

Before installing project, be sure to have installed all dependencies and apps to run React Native Expo project on your device.

##### Install all plackages from package.json:

```js
npm i
```

##### OR

```js
yarn;
```

run the project:

```js
yarn start
```

## Package Used

1. Axios
2. Expo
3. React Native async storage
4. React Navigation
5. React Native Maps
